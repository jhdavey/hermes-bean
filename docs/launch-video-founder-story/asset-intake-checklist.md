# Asset Intake Checklist

Put everything you create or collect under:

`docs/launch-video-founder-story/assets/from-josh/`

## Required

### 1. Voiceover

Folder:

`assets/from-josh/voiceover/`

Please provide:

- One full voiceover recording of the script in `voiceover-script.md`
- Preferred formats: `.wav`, `.m4a`, or `.mp3`
- File name suggestion: `heybean-founder-story-voiceover.wav`

Recording guidance:

- Record in a quiet room.
- Speak naturally and a little slower than normal conversation.
- Leave pauses between paragraphs.
- If you make a mistake, pause and restart the sentence.
- Do not add music or effects.

### 2. Product Screen Recordings

Folder:

`assets/from-josh/screen-recordings/`

Please provide clean recordings of:

- Home or Today view
- Calendar view
- Tasks or reminders view
- Assistant/chat view
- A short sequence showing the assistant helping with planning, priorities, calendar, tasks, or reminders

Recording guidance:

- Use demo data only.
- Avoid private names, real addresses, phone numbers, emails, personal events, or sensitive reminders.
- Record at the highest resolution available.
- Slow, deliberate interactions are better than fast clicking.
- Hide browser chrome if possible.
- If recording mobile, use the simulator or a clean screen recording.

Suggested file names:

- `01-home-today-view.mp4`
- `02-calendar-view.mp4`
- `03-tasks-reminders-view.mp4`
- `04-assistant-planning-flow.mp4`
- `05-full-product-flow.mp4`
